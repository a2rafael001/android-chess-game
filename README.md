# Android Chess Game — игра в шахматы для Android

Небольшое учебное приложение на **Android**, реализующее простую игру в шахматы.  
Проект собран в **Android Studio** на языке **Kotlin** (по данным GitHub — 100% Kotlin).  

Основная цель проекта — показать умение:

- работать с Android-проектом “под ключ” (Gradle, модули `app`, `gradle` и т.д.);
- верстать интерфейсы под Android (layout’ы, ресурсы, строки);
- писать базовую игровую логику.

---

## Возможности приложения

> Проект учебный, функциональность может развиваться по мере выполнения лабораторных.

На текущем этапе реализовано:

- Игровое поле **8×8**, оформленное как шахматная доска.
- Отображение шахматных фигур на клетках.
- Выбор клетки тапом по экрану.
- Перезапуск партии (новая игра) через кнопку / пункт меню.
- Отображение интерфейса в портретной ориентации.

Фокус сделан на **структуре Android-приложения и UI**, а не на “идеальном” движке шахмат.  
Поэтому часть логики (шах, мат, рокировка, ИИ и т.п.) может быть реализована частично либо отложена в “план доработок”.

---

## Технологии

- **Язык:** Kotlin  
- **Платформа:** Android  
- **IDE:** Android Studio  
- **Сборка:** Gradle (`build.gradle.kts`, `gradle.properties`, `settings.gradle.kts`)  
- **UI:** классическая Android View-система и XML-разметка:
  - `FrameLayout` — контейнеры для ячеек/элементов;
  - `GridLayout` — сетка 8×8 для шахматной доски;
  - другие стандартные layout’ы (например, `LinearLayout` для панелей).

---

## Структура проекта

В корне репозитория:

- `app/` — основной модуль Android-приложения:
  - `src/main/java/...` — исходный код (активити, логика доски и т.п.);
  - `src/main/res/layout/` — XML-разметка экранов;
  - `src/main/res/drawable/` — ресурсы (иконки фигур / фоны);
  - `src/main/res/values/strings.xml` — строковые ресурсы;
  - `src/main/AndroidManifest.xml` — описание приложения и точка входа.
- `gradle/` — файлы Gradle-wrapper’а.
- `build.gradle.kts`, `gradle.properties`, `settings.gradle.kts` — конфигурация проекта и модулей.
- `gradlew`, `gradlew.bat` — скрипты для запуска Gradle из консоли.
